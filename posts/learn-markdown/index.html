Markdown

1. 标题

1-6个#
快捷键：Ctrl(Command) + 1-6
清除标题：Ctrl(Command) + 0

一级标题 # 一级标题

二级标题 ## 二级标题

三级标题 ### 三级标题

四级标题 #### 四级标题

五级标题 ##### 五级标题
六级标题 ###### 六级标题

2. 文字标记

  1. 粗体： Bold **Bold**
  2. 斜体： Italic *Italic*
  3. 粗斜体： Bold Italic ***Bold Italic***
  4. 中划线： Inner Line ~~Inner Line~~
  5. 下划线： Underline _Underline_
  6. 中划线下划线： _Innder Under List_ ~~_Innder Under List_~~
  7. 高亮显示： I will learn ==Python== later. I will learn ==Python== later.

4. 无序列表

* - +
"*"、"-"、"+"后面加空格
无序列表可以和有序列表一起使用
退出列表：行尾 两个Enter键，或⬇️
增加缩进：Tab
减少缩进：Shift + Tab

  • List Two with *
  • list 1 nesting Two
  • list 2 nesting Two
  • List Three with +
  • List Four with -
  • List nesting Four
